# Documentação da API

## Endpoints principais (exemplo)
- `GET /api/v1/servidores` — Lista servidores
- `POST /api/v1/servidores` — Cria servidor
- `GET /api/v1/pacientes` — Lista pacientes

## Autenticação
JWT Bearer Token

## Formato de resposta
```json
{
  "success": true,
  "data": [...],
  "message": ""
}
```
